# 钉钉适配器

钉钉适配器（计划中）将支持通过钉钉机器人接入 OneBots 服务。

## 状态

🚧 计划中

## 计划支持

- 钉钉企业内部机器人
- 钉钉群机器人
- 消息收发
- 卡片消息
- 事件订阅

## 开发计划

钉钉适配器正在规划中，预计将支持：

1. **企业内部应用机器人**
   - 通过企业内部应用接入
   - 支持完整的消息和事件能力

2. **自定义机器人**
   - 通过 Webhook 方式接入
   - 支持群聊消息推送
