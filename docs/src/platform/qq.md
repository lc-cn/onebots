# QQ 适配器

QQ 适配器（计划中）将支持通过 QQ 官方 API 接入 OneBots 服务。

## 状态

🚧 开发中

## 计划支持

- QQ 频道机器人
- QQ 群机器人（官方 API）
- 消息收发
- 群组管理
- 成员管理

## 替代方案

目前如需接入 QQ，可使用社区的第三方实现：

- [icqq](https://github.com/icqqjs/icqq) - QQ 协议实现
- [NapCat](https://github.com/NapNeko/NapCatQQ) - OneBot V11/V12 协议实现

## 开发计划

我们正在评估官方 API 的能力，计划在官方 API 稳定后提供官方适配器。
