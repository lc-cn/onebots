# Kook 适配器

Kook（原开黑啦）适配器（计划中）将支持通过 Kook Bot API 接入 OneBots 服务。

## 状态

🚧 计划中

## 计划支持

- Kook Bot 机器人
- 消息收发（文本、图片、卡片等）
- 频道管理
- 成员管理
- 语音频道事件

## 开发计划

Kook 适配器正在规划中，将基于 Kook 官方 Bot API 开发。
