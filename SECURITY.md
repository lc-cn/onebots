# 安全政策

## 支持的版本

我们目前支持以下版本的安全更新：

| 版本   | 支持状态        |
| ------ | --------------- |
| 0.4.x  | :white_check_mark: |
| < 0.4  | :x:             |

## 报告漏洞

如果您发现了安全漏洞，请**不要**公开提交 Issue。

请通过以下方式报告：

1. 发送邮件到项目维护者
2. 在 GitHub 上使用 Security Advisory 功能（推荐）

我们会在 48 小时内回复您的报告，并在确认后尽快发布修复。

### 报告应包含的信息

- 漏洞类型
- 受影响的版本
- 复现步骤
- 潜在影响
- 修复建议（如果有）

感谢您帮助保护 OneBots 用户的安全！
